## Getting Started

To fork and enhance this New Relic One application, follow the [Getting Started instructions in the README](https://github.com/newrelic/nr1-cloud-optimize#getting-started).

## Features

- Analyze your cloud environment using the New Relic Infrastructure cloud integrations
- Compares the size of your instances to their utilization
- Identify resources that are sized larger than needed
- Estimate your savings by optimizing resource size
- Select the hosts, regions and other configurations to specify unique business use cases
- Supports AWS, GCP, and Azure cloud infrastructures
