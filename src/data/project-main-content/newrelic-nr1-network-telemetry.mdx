The Network Telemetry application displays the data collected by the [Network Telemetry Integration](/projects/newrelic/nri-network-telemetry) for New Relic Infrastructure.

## Getting Started

Workload GeoOps is available for installation via the [New Relic One Catalog](https://docs.newrelic.com/docs/new-relic-one/use-new-relic-one/build-new-relic-one/discover-manage-applications-new-relic-one-catalog).

## Features

- Visualize data captured by the [Network Telemetry Integration](/projects/newrelic/nri-network-telemetry)
