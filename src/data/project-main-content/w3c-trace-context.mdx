## How New Relic is Involved

New Relic is a participant in the [W3C Distributed Tracing Working Group](https://www.w3.org/2018/distributed-tracing/).

All of the New Relic agents implement the [W3C TraceContext specification](https://github.com/w3c/trace-context/).
