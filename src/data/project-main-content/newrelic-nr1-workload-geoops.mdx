Workloads GeoOps provides a geographic exploration of [Workload](https://docs.newrelic.com/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-one-workloads-isolate-resolve-incidents-faster) data (collections of Entities) tied to geographic data, including location metadata. Find additional information in the [docs](https://github.com/newrelic/nr1-workload-geoops/blob/master/docs/documentation.md) directory.

## Getting Started

Workload GeoOps is available in the New Relic One Catalog.

<Link to="https://one.newrelic.com/launcher/nr1-catalog.launcher">Install</Link>

## Features

- Align Workloads and Entities to geographic data
- Observe rollup alerting state across Entities tied to location
- Drive into Workload and Entity workflows in New Relic One
- Configure via the point-and-click or my uploading JSON configs
- Use the [New Relic CLI](https://github.com/newrelic/newrelic-cli) to [automate configuration](https://github.com/newrelic/nr1-workload-geoops/blob/master/docs/AUTOMATION.md)
